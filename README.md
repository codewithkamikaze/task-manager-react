A lightweight task manager built with **React**, using **IndexedDB** for local storage. Manage your tasks efficiently with filtering, search, and quick shortcuts.

---

## Features

- Add, edit, and delete tasks
- Mark tasks as **Pending** or **Done**
- Filter tasks by status (`All`, `Pending`, `Done`)
- Search tasks by title or description
- Seed sample tasks for quick start
- Clear all tasks at once
- **Ctrl+Enter / Cmd+Enter** shortcut for quick task addition
- Persistent storage using **IndexedDB**
- Responsive UI with TailwindCSS

---

## Getting Started

### Prerequisites

- Node.js ≥ 16
- npm or yarn

### Install Dependencies

```bash
npm install
# or
yarn install
```
